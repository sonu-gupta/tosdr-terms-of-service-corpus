The maximum weight allowed for each item shipped is 30 kg (per package). 

The seller must be careful to select the correct package size in order to avoid additional costs from Correos. 

So that the item arrives in the best conditions possible, it is recommendable to package it using specialised envelopes or padded envelopes. It is important to prepare the item carefully for shipping so that it does not suffer any damage during the transit process. 

Wallapop will not be responsible in any case for damages caused as a consequence of incorrect or insufficient packaging. 

Taking the package to Correos

The time limit to take the package to Correos is five calendar days from the date of the sale of the item. After this time limit of five days the code for Correos will no longer be valid and the amount will be automatically returned to the buyer. 

It’s very important to respect the time limit: if it is not respected the transaction will be cancelled and the buyer will have to repeat the whole transaction process.

Once the seller has accepted the payment we will send them a shipping code which will allow them to check the status of the delivery at any time. It is also possible to check the status of your package via the package locator on the Correos website.

The buyer has 3 business days to verify that the item is in working order.
