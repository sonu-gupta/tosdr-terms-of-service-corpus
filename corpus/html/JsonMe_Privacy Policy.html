Privacy policyData collection

The only data that JsonMe collects is your Github/Gitlab username. Nothing else. The rest of the data is consumed from Github/Gitlab directly so you have full control over it.

Cookies

This website uses two cookies: One required by the web application framework that it is using. No user information is stored inside. The other one is used by the web analytics software that is used. You can read more about it in the next section.

Web analytics

JsonMe uses privacy-focused self-hosted website analytics - Umami. It does not collect any personally identifiable information and anonymizes all data collected.
