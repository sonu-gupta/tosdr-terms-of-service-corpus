This statement which updated on July 18, 2011 will take effect immediately.

Your privacy will be respected.
Sometimes we need some information to give you a better user experience.This statement informed about how we collect and use your information.
Make sure your personal information is better protected.

  1. collection of your personal information

    当我们需要能识别您身份的信息或您的联系方式时，我们会征求您的同意。通常，在您注册又拍图片管家或者通过又拍图片管家访问任何第三方合作方时，我们会需要您提供这些信息。

  2. use of personal information

    又拍图片管家仅将您的信息用于对您服务的支持上，并且保证信息的使用完全符合中华人民共和国相关的法律法规。 又拍图片管家不会未经您允许将您的信息用于任何第三方平台上。

  3. protection of personal information

    又拍图片管家严格保护您的个人信息的安全。我们使用各种安全技术和程序来保护您的个人信息不被未经授权的访问、使用或泄漏。如果您对我们的隐私保护有任何置疑，可发邮件至faq#yupoo-inc.com（用@替换#），我们将尽快为您处理。

  4. other technologies

    我们将在您访问我们的 Web 站点时收集信息，以帮助我们统计相关的使用情况和数据，以提供更好的用户体验。

  5. cookie,web beacon and other technologies

    我们通过包括cookie在内的技术来收集信息。

    Cookie常用于衡量 Web 站点的使用情况和效率， 也被用于个性化已知访客对 Web 站点的体验，且它与个人信息没有任何关联。您也可以对浏览器进行设置以关闭 Cookie，但可能会影响到我们 Web 站点一些功能的使用，我们建议您将其打开。

    又拍图片管家还可能使用 Web beacon等技术以便更好地调整站点以提供更好的服务。当浏览者访问这些页面时，就会产生该访问的一个匿名通知，由我们或我们的供应商来处理。这些 Web beacon 通常与 Cookie 一起协同工作。如果您不希望把您的 Cookie 信息和您对这些页面的访问联系在一起的话，您可以对浏览器进行设置以关闭 Cookie，关闭后，，Web beacon 和其它技术仍然会检测对这些页面的访问；但它们将不会与存储在 Cookie 中的信息有任何联系。

    我们可能会在市场推广电子邮件或实时通讯中使用Web beacon 技术以确定邮件中包含的链接是否已被点击或者消息是否已被打开。

    我们的部分业务伙伴在我们站点中也设置了 Web beacon 和 Cookie。我们无权访问或控制此类第三方的跟踪技术。

  6. About exemption

    就下列相关事宜的发生，又拍图片管家不承担任何法律责任：

    1. 又拍图片管家根据法律规定或相关政府的要求提供您的个人信息；
    2. 由于您将用户密码告知他人或与他人共享注册帐户导致的任何个人信息的泄漏，其他非又拍图片管家导致的个人信息的泄漏；
    3. 任何第三方根据又拍图片管家各服务条款及声明中所列明的情况使用您的个人信息，由此所产生的纠纷；
    4. 任何由于黑客攻击、电脑病毒侵入或政府管制而造成的暂时性网站关闭；
    5. 因不可抗力导致的任何后果；
    6. 又拍图片管家在各服务条款及声明中列明的使用方式或免责情形。
  7. Highlights
    1. “又拍图片管家”系提供信息存储空间的网络服务提供者，在“又拍图片管家”平台上的信息（包括但不限于图片、联系方式等信息）均由网络用户自行上传，网络用户应当自行对其提供任何信息承担全部法律责任。
    2. 网络用户在“又拍图片管家”平台上的所有行为均应遵守中国人民共和国相关法律法规及政策的规定，不得利用本平台进行法律、法规和政策禁止的内容。“又拍图片管家”有权根据自己的合理判断，决定对网络用户采取停止服务、删除信息、禁号、不退平台服务款、追究法律责任等措施，上述措施不限于一项。
    3. “又拍图片管家”既尊重他人的知识产权也注重保护网络用户的合法权益。任何单位或个人认为“又拍图片管家”平台网页内容涉嫌侵犯其合法权益的，应该及时向平台管理人员提出书面的权利通知，并提供身份信息证明、权属证明及明确的网络链接地址。平台管理人员收到相关材料后及时进行审核并尽快予以处理。
    4. 任何由于黑客攻击、电脑病毒侵入或政府管制而造成的暂时性网站关闭；

    又拍图片管家会不定时更新本隐私权声明。届时，我们将会更新隐私权声明顶部的"最近更新"的日期。
